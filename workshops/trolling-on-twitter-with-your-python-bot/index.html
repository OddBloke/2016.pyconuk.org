Trolling on Twitter with your Python bot

Carlos Edo | Thursday 09:30 | CU Room B

Using Python and Twitter's API we will create an "intelligent" bot. The bot will make searches for a predefined text and post clever tweet answers. The script will be uploaded to Heroku and, using the "Heroku Scheduler", we will automate its execution.
